# HelpDeskSystem
Role-based ticket management system built with ASP.NET Core MVC using Identity, EF Core, and SQL Server.

EN

Mini HelpDesk is a small-scale ticket management web application developed using ASP.NET Core MVC.
The system allows users to create support tickets and track them, while administrators can manage and respond to those tickets through an admin panel.

The application demonstrates the core concepts of modern .NET web development, including MVC architecture, authentication, role-based authorization, and database operations.

Features

User registration and login using ASP.NET Core Identity

Role-based authorization (Admin / User)

Create, view, and manage support tickets

Admin panel for managing tickets and categories

Ticket status management (Open / Closed)

Admin response to user tickets

Category filtering

Form validation using FluentValidation

Technologies

ASP.NET Core 8.0 MVC

Razor Views

Entity Framework Core

ASP.NET Core Identity

SQL Server

FluentValidation

Bootstrap 

TR

Mini HelpDesk, ASP.NET Core MVC kullanılarak geliştirilmiş küçük ölçekli bir ticket (destek talebi) yönetim web uygulamasıdır.

Sistem, kullanıcıların destek talebi oluşturmasını ve taleplerini takip etmesini sağlarken, admin kullanıcıların bu talepleri admin paneli üzerinden yönetmesine ve cevaplamasına olanak tanır.

Bu proje, modern .NET web uygulaması geliştirme süreçlerini öğrenmek amacıyla hazırlanmıştır.

Özellikler

ASP.NET Core Identity ile kullanıcı kayıt ve giriş sistemi

Rol bazlı yetkilendirme (Admin / User)

Kullanıcıların ticket oluşturabilmesi ve görüntüleyebilmesi

Admin paneli üzerinden ticket yönetimi

Ticket durum yönetimi (Açık / Kapalı)

Admin tarafından ticket cevaplama

Kategori bazlı filtreleme

FluentValidation ile form doğrulama

Kullanılan Teknolojiler

ASP.NET Core 8.0 MVC

ASP.NET Core Identity
